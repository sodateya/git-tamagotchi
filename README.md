# 🥚 Git Tamagotchi

GitHub の活動（コミット・PR・レビュー・Issue）でデスクトップ上のペット／植物が育つ、常駐型の育成ガジェットです。エンジニアの成果を「お世話」に変換して、楽しく成果を出すためのツール。

```
コミット    = エサ／水やり   （2pt）
PR 作成     = 成長イベント   （8pt）
レビュー    = 特別ボーナス   （12pt・他者貢献を厚く評価）
Issue 対応  = おやつ         （4pt）
```

累計ポイントでステージが上がり、直近の活動状況で「元気度」が変化します。活動が続くと🔥連続日数が伸び、しばらく放置するとお腹を空かせます。

| ステージ | ペット | 植物 |
|---|---|---|
| 0pt | 🥚 たまご | 🌰 たね |
| 50pt | 🐣 ひな | 🌱 め |
| 200pt | 🐥 こども | 🌿 わかば |
| 600pt | 🐔 おとな | 🌷 かいか |
| 1500pt | 🦅 達人 | 🌳 大樹 |

## 必要なもの

- Node.js 18 以上（推奨: 20 以上）
- GitHub アカウント

## セットアップ

```bash
# 1. 依存をインストール
npm install

# 2. 起動
npm start
```

起動するとデスクトップに小さな筐体が浮かび、タスクトレイ（メニューバー）にアイコンが出ます。

## GitHub トークンの設定

1. トレイアイコン → 「設定を開く…」
2. 設定画面の「トークンを発行する →」リンクから GitHub のトークン作成ページへ
3. 権限は **read:user** のみでOK（fine-grained トークンなら読み取り専用で十分）
4. 発行したトークンを貼り付けて「保存する」

トークンは外部に送信されず、`~/.git-tamagotchi/config.json` にこのPC内だけで保存されます。

## 操作

- **筐体をドラッグ**: 好きな位置へ移動
- **↻ ボタン**: いますぐ最新の活動を取得（自動でも30分ごとに更新）
- **@ ボタン**: 自分の GitHub プロフィールを開く
- **トレイメニュー**: ペット／植物の切り替え、更新、設定、終了

## ファイル構成

```
git-tamagotchi/
├── main.js            Electronメイン（ウィンドウ/トレイ/定期更新）
├── preload.js         レンダラーへ安全にAPIを公開
├── src/
│   ├── github.js      GitHub GraphQL でアクティビティ取得
│   ├── growth.js      育成ロジック（スコア/ステージ/元気度）← 数値はここで調整
│   ├── index.html     キャラ表示（たまごっち風LCD）
│   ├── renderer.js    キャラ描画の更新
│   ├── settings.html  設定画面
│   └── settings.js    設定ロジック
```

## カスタマイズの入り口

- **ポイント配分やステージのしきい値**: `src/growth.js` の `POINTS` と `STAGES`
- **キャラの見た目（絵文字）**: `src/growth.js` の `STAGES[].emoji`
- **元気度の判定**: `src/growth.js` の `streakAndMood`
- **筐体の色・アニメーション**: `src/index.html` の `<style>`

## 今後の発展アイデア

- PR の「マージ済み」を別カウントして大きな成長イベントに（GraphQL の検索APIを追加）
- リポジトリ別／言語別に違う種類のペットを育てる
- 連続日数のごほうび進化、レアな見た目の解放
- 複数キャラを並べてチームの活動を表示
- 配布したい場合は electron-builder でインストーラ化（その際はトークンを OAuth 方式に変更推奨）

## 注意

- これは個人用ローカルツールの想定です。配布する場合はトークンの平文保存をやめ、OS のキーチェーンや OAuth デバイスフローへの変更を推奨します。
